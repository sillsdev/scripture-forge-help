Segment

In Scripture Forge kann ein Segment ein Bibelvers oder ein Teil eines Verses sein. Zum Beispiel könnte ein Vers einen Text, ein Zitat und dann mehr Text haben. In diesem Fall würde der Vers aus drei Abschnitten bestehen.

Wenn Administratoren den Fortschritt überprüfen, finden sie die Anzahl der Verse (Segmente), die Text enthalten (Übersetzung) und die Gesamtzahl der Segmente in jedem Buch. Eine kreisförmige Fortschrittsanzeige bietet eine grafische Darstellung dieser Zahlen.
