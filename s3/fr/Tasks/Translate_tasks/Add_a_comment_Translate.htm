Ajouter un commentaire à un verset

Ce sujet est terminé dans la zone Traduire. Les personnes ayant le rôle d'Admin, de Traducteur ou de Commentateur peuvent ajouter ces commentaires.

Si vous êtes un Vérificateur communautaire, consultez : Ajouter un commentaire pour ajouter un commentaire à une réponse.

  1. Choisissez un projet.

  2. Dans le volet Navigation, cliquez sur Traduire.

  3. Cliquez sur le livre ( <name>).

  4. Cliquez sur le verset sur lequel vous souhaitez commenter.

  5. Cliquez sur .

Une boîte de dialogue apparaît, avec le livre, le chapitre et le verset affichés dans la barre de titre.

  • Vérifiez s'ils sont corrects.

  1. Tapez votre commentaire dans la boîte de dialogue, puis cliquez sur Enregistrer.

Ce verset affiche maintenant une icône de commentaire ().

  • Placez votre curseur de souris sur cette icône pour lire le commentaire.

Si vous utilisez un téléphone ou un appareil plus petit, appuyez sur l'icône pour ouvrir la boîte de dialogue afin de pouvoir le lire.

Modifier un commentaire

  1. Cliquez ou appuyez sur l'icône () pour ouvrir à nouveau la boîte de dialogue.

  2. ou appuyez sur le crayon d'édition ( ) pour le mettre en mode édition.

  3. Effectuez les modifications souhaitées.

  4. Cliquez sur Enregistrer. 

Remarque

  • Sur un appareil mobile, une feuille inférieure apparaît que vous pouvez utiliser pour entrer et enregistrer un commentaire.

  • Lorsque les commentaires sont exportés vers PT sous forme de note, le nom du commentateur est affiché comme une étiquette sur la note.
