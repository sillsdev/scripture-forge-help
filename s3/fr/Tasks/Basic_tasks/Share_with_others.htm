Partager avec d'autres

Les administrateurs peuvent toujours inviter d'autres personnes à partager le travail dans le projet. Les personnes ayant d'autres rôles peuvent également inviter d'autres personnes, mais les options de rôle qu'elles peuvent sélectionner peuvent être différentes.

Les personnes que vous invitez ne pas doivent avoir un compte Paratext. Les personnes qui ont un compte Paratext n'ont pas besoin d'être invitées

Si l'administrateur a activé la fonctionnalité Partager, alors les autres utilisateurs peuvent faire ces étapes :

  1. Dans le volet Navigation, cliquez sur un livre sous Traduire ou Vérification par la communauté.

  2. Cliquez sur(Partager) sur la barre d'outils .

La boîte de dialogue Partagerapparaît.

  1. Cliquez sur changez pour n'importe quelle option (,ou ) jusqu'à l'invitationrôle, langue du rôle d'invitation, et qui peut utiliser le lien montrer ce que vous voulez faire.

  2. Effectuez l'une de ces étapes :

  • Sur un ordinateur de bureau ou un ordinateur portable, cliquez surCopier le lien pour le copier dans votre presse-papiers.

Ensuite, vous pouvez coller ce lien dans un message texte ou de chat, poster dans Facebook ou dans un e-mail, et ainsi de suite.

  • Si vous êtes sur un appareil mobile, appuyez sur le boutonPartager. Choisissez ensuite la meilleure option à votre disposition pour envoyer cette invitation aux autres.

Important

  • Si Activer la vérification de la communauté n'est pas sélectionné () dans Paramettres, le seul rôle disponible de Scripture Forge est Observateur.

  • Une fois que le projet est connecté, tout autre utilisateur de Paratext de ce projet peut se connecter dans Scripture Forge.

Cependant, les personnes qui feront Vérification de la Communauté sont et non généralement des utilisateurs de Paratext. Dans ce cas, Activer la vérification de la communauté doit être sélectionné ().

  • La personne invitée doit s'inscrire pour obtenir un compte si elle n'en a pas déjà un.

  • Les administrateurs peuvent faire ces mêmes étapes dans Utilisateurs.
