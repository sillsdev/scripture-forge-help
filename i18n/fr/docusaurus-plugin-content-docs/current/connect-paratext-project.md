Avant de pouvoir accéder à un projet Paratext dans Scripture Forge, il doit être connecté à Scripture Forge. La connexion d'un projet crée une copie du projet Paratext dans Scripture Forge qui peut être [synchronisée avec Paratext](syncing-with-paratext).

Dans Paratext, chaque membre du projet a une copie du projet sur son propre ordinateur, et Envoyer/Recevoir est utilisé pour partager les changements avec d'autres membres. Quand un projet est connecté à Scripture Forge, tous les membres du projet dans Scripture Forge travaillent sur une seule copie du projet, de sorte que tout changement effectué par un membre est immédiatement vu par un autre membre, si les deux membres sont en ligne. Ces modifications seront envoyées au projet Paratext quand une synchronisation est effectuée dans Scripture Forge, et toutes les modifications apportées dans Paratext seront synchronisées avec le projet Scripture Forge.

Pour connecter un projet Paratext dans Scripture Forge, [connectez-vous avec Paratext](log-in). Lorsque vous vous connectez, la page **Mes projets** se chargera. Si vous êtes sur une page différente dans Scripture Forge, vous pouvez accéder à la page **Mes projets** en cliquant sur le logo de Scripture Forge dans le coin supérieur gauche de la page. La page **Mes projets** répertorie tous les projets dont vous êtes membre dans Paratext, y compris ceux qui sont connectés à Scripture Forge et ceux qui ne le sont pas encore. Si un projet a déjà été connecté et que vous êtes membre du projet Paratext, vous pouvez rejoindre le projet Scripture Forge en cliquant sur **Join**. Si le projet n'a pas encore été connecté et que vous êtes administrateur du projet, cliquez sur **Connecter**. Seuls les administrateurs peuvent connecter un projet sur cette page.
